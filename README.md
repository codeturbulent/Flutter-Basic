# Basic Flutter - Learning Collection

This repository contains a collection of Flutter and Dart projects developed during my initial learning phase. These projects cover various features, from camera integration to web scraping and basic UI implementations.

## 📂 Projects in this Repository

1. **[flutercamapp](./flutercamapp)** - A camera integration app using the `camera` plugin.
2. **[ocrinanapp](./ocrinanapp)** - An OCR (Optical Character Recognition) application.
3. **[onbording](./onbording)** - Simple and interactive onboarding screens.
4. **[testai](./testai)** - Exploring AI integration with Flutter.
5. **[vcdemo](./vcdemo)** - Video conferencing and communication demo.
6. **[webscrap](./webscrap)** - Web scraping implementation in Dart/Flutter.
7. **[webserverstarter](./webserverstarter)** - A basic Dart-based web server setup.

## 🛠️ Common Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)
- Android Studio / VS Code with Flutter extension

## 🚀 How to Run any Project
1. Navigate to the project directory:
   ```bash
   cd <project_name>
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
