# Flutter Camera App

A Flutter application demonstrating the implementation of device camera features.

## ✨ Features
- **Real-time Camera Preview:** View camera output in real-time.
- **Image Capture:** Capture and display high-quality photos.
- **Multi-Platform Support:** Ready for Android and iOS.

## 📦 Dependencies
- \`camera\`: For camera access and control.
- \`path_provider\`: For managing file storage.
- \`path\`: For handling file paths.
